# Hi Sora

Restaurant website, online ordering, and digital growth site for Hi Sora.

## Scripts

- `pnpm dev` starts the local Astro development server.
- `pnpm check` runs Astro type diagnostics.
- `pnpm build` builds the Cloudflare production output.

## Production

The public site is configured for `https://hisora.cc` and deploys through Cloudflare.
